动机也不评价。

既然题主你问如何反驳，那就直接反驳呗。

  • 可以多任务并行：Thread support library。
  • 最简单的编程语言：770+页的 Java Language Specification，而包含了标准库的 C11 标准也只有696页。
  • 最好的编程语言：Is PHP the best programming language?


给题主的一个建议：要想反驳对方的话肯定要先了解对方所陈述的内容和对方所站的论点。换句话说在这个场景中你就得对 Java、C、并行编程以及 PHP 都有一定的了解才行。

不然完全驴唇不对马嘴你跟他争个茄子啊。
